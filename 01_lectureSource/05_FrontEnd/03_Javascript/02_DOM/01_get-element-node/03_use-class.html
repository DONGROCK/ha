03. class를 이용한 요소 노드 취득

        
            Document.prototype/Element.prototype.getElementsByClassName 메서드는
            인수로 전달한 class 어트리뷰트 값을갖는 모든 요소들을 탐색하여 반환한다.
        
        메서드 이름의 Elements 가 복수형인 것에서 알 수 있듯 여러 개의 요소 노드 객체를
        갖는 DOM컬렉션 객체인 HTMLCollection객체를 반환한다.
