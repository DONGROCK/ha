                Document.prototype/Element.prototype.querySelectorAll 메서드는
                인수로 전달한 CSS선택자를 ㅁ나족시키는 모든 요소 노드를 탐색하여 반환한다.
            
            여러개의 요소 노드객체를 갖는 DOM컬렉션 객체인 NodeList객체를 반환한다.
